Lazy

A chrome extension for the lazily efficient people.

How it works

This extension will wait until the TAB key is pressed in a FS-UK textarea (forum post box, new topic box, reply box, etc).

When the key is pressed, it will look for the word right before the cursor, the trigger, and will see if it matches any entry in the table below.

If it does, the word right before the cursor gets replaced by the matching code value for that snippet.
